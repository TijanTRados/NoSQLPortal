Projekt: NoSQL Projekt
Kolegij: Napredni Modeli i Baze Podataka
Autor: Tijan Tomislav Radoš (0036462866)

Zadatak: Napraviti jednostavan portal s vijestima današnjice korištenjem NoSQL baze podataka uz MongoDB, ostvariti moguænost
komentiranja.

Korištene tehnologije:
	MongoDB
	Visual Studio 2015 (C#) ASP.NET MVC aplikacija

U programskom okruženju Visual Studio 2015, koristeæi programski jezik C# ostvario sam sljedeæe moguænosti:
	- pregled Web portala (server = localhost) - prikaz zadnjih 10 najnovijih vijesti poredanih od najnovijih na dalje
	- unos nove vijesti (maska na dnu web stranice)
	- unos pozadinske fotografije za odreðenu vijest
	- komentiranje vijesti

Upute za instalaciju i pokretanje:
	(zahtjeva Visual Studio)
	Unutar programskog okruženja Visual Studio otvoriti solution "Portal" ili samo pokrenuti solution "Portal" uz prethodno instaliran Visual Studio
